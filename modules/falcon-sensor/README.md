# CrowdStrike Falcon Agent Sensor

## How To

Compile and run with

```shell
nix-build
result/bin/falcon-sensor
```

## Alternatives

- [distrobox](https://github.com/89luca89/distrobox/tree/main)
- Docker
- steam-run
